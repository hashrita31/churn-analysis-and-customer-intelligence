# churn-analysis-and-customer-intelligence
Customer Churn Analysis &amp; Customer Intelligence is a Python and SQL project that analyzes customer behavior to identify churn patterns and business insights. It includes data cleaning, EDA, KPI analysis, customer segmentation, SQL queries, and visualizations using Pandas, NumPy, Matplotlib, Seaborn, and Jupyter Notebook
